# 乒乓球计分器

## 项目简介

智能乒乓球计分系统是一款基于计算机视觉和语音识别技术的创新型应用，旨在为乒乓球比赛提供自动化的计分和实时反馈功能。



### 语音识别体验升级

- **语音识别速度大幅提升**：录音时长缩短为0.8秒，识别响应更快。
- **中英文并行识别**：同时请求中文和英文，谁先识别到得分指令即刻加分，识别率更高。
- **语音进度条可视化**：界面中央新增大号进度条，实时显示录音进度、识别状态和得分反馈。
  - 录音时进度条线性增长，显示“录音中...”
  - 识别时橙色跑马灯动画，显示“识别中...”
  - 得分成功时绿色满条高亮，显示“A/B +1 !”
  - 空闲时灰色底条+绿色空心圆点
- **录音参数更灵敏**：极短停顿即视为说完，适合快速喊分。
- **视觉反馈更直观**：所有语音状态均有清晰中文提示和动态进度条。

### 其他优化

- 语音识别冷却期、发球方切换、比赛规则等逻辑进一步完善。
- 进度条、得分动画、发球提示等UI细节优化。

## 开发计划

1. 增加更多的手势识别类型（如挥拍动作）。
2. 提供更灵活的语音关键词配置，支持本地离线语音识别。
3. 增加网络功能，支持远程计分和观赛。
4. 添加比赛回放和统计功能（得分时间轴、胜率分析等）。
5. 优化语音识别的背景噪音处理。
6. 支持自定义赛制（如21分制、局数配置等）。
